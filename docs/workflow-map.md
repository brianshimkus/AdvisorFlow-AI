# Workflow Map

## Manual workflow (today)

```
Advisor reviews CRM
  -> reads last meeting notes
  -> checks tasks and account activity
  -> skims internal playbook
  -> writes prep summary
  -> identifies follow-up questions
  -> prepares for the client meeting
```

## AI-assisted workflow (AdvisorFlow AI)

```
Advisor selects client
  -> system retrieves CRM, notes, tasks, and playbook context
  -> AI generates a source-grounded prep brief
  -> advisor reviews citations, risks, and missing-data warnings
  -> advisor edits/exports final brief
  -> system logs useful debug and evaluation information
```

## Step-by-step breakdown

Review CRM:
Pain point - time-consuming context search across scattered records
AI-assisted replacement - pull in client profile automatically

Read notes:
Pain point - important details can be buried in long note threads and missed
AI-assisted replacement - provide notes with important and relevant details, with citations

Check tasks:
Pain point - open follow-ups are scattered and easy to lose track of
AI-assisted replacement - evaluate outstanding and overdue tasks automatically

Skim playbook:
Pain point - guidance is applied inconsistently from advisor to advisor
AI-assisted replacement - pull in relevant playbook snippets

Write prep doc:
Pain point - repeats every meeting and prep quality varies by person
AI-assisted replacement - generate a first draft to be reviewed by a human
